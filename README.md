 # Backend
- Express - HTTP Server
- mongoose - ODM to connect to MongoDB
- zod - Input validation
- mongo db - database
- bcrypt - For hashing password before storing it in db
- jsonwebtokens - for generating tokens and token validation
- cors - for avoiding browser cors error
